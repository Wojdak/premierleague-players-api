Football Players API

The API offers information on players, clubs, nationalities, and positions from the English Premier League.
Deployed as an Azure Web App, it uses Azure SQL Database for data storage.

*The initial request may take longer than expected due to the serverless nature of the database, which pauses when not in use for an extended period.
