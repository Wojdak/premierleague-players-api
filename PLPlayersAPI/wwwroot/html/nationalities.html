GET /api/nationalities Get a list of nationalities

Get a list of nationalities

Resposne Format
{
    "nationalityId": 1,
    "country": "Albania",
    "flagSrc": "https://resources.premierleague.com/premierleague/flags/AL.png"
},
{
    "nationalityId": 2,
    "country": "Algeria",
    "flagSrc": "https://resources.premierleague.com/premierleague/flags/DZ.png"
}
Response Code

200 OK - with list of nationalities

GET /api/nationalities/{Id} Get a nationality with the given ID

Get a nationality with the given ID

Parameters
  • Id (int): Unique identifier of the nationality
Example Usage
GET /api/nationalities/1
Response Format
{
    "nationalityId": 1,
    "country": "Albania",
    "flagSrc": "https://resources.premierleague.com/premierleague/flags/AL.png"
}
Response Codes

200 OK - with one nationality
404 Not Found - if nationality with the given ID doesn't exist in the database

POST /api/nationalities Add new nationality

Add new nationality

Authorization (JWT Token) Required

This endpoint is only available for authorized users

Validation Rules
  • Minimum length of the country is 3 characters
  • The flagSrc can be null
Request Body
{
    "country": "string"
    "flagSrc": "string"
}
Response Format

Successfully added a new nationality with id: {nationalityId}

Response Codes

201 Created - with a response message
400 Bad Request - if nationality didn't pass validation
